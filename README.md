    - ls -al

    - rmdir   => remove directory

    - unlink --> file remove

    - man

    - cd   --> change directory

    - cat file_name  (concatenate)
    - cat file_name_1 file_name_2

    - .. --> bir üst klasörde 
    - .  --> current directory
